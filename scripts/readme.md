# Scenarijai

Aplinkos scenarijai (shell scripts) skirtingomis programavimo kalbomis ar jų dialektais. Tikslas - patikrinti, kaip skirtingos kalbos tinka rašyti operacijų sistemos valdymo scenarijus.

* [sys-upgrade](./sys-upgrade/readme.md "sys-upgrade") - Ubuntu paketų atnaujinimo scenarijus.
